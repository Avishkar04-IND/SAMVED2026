# SafeGuard — Real-Time IoT Safety System for Sanitation Workers

**Team Phoenix_26 | SAMVED Hackathon 2026 | MITVPU, Pune**  
**Mentor:** Prof. Prateek Meshram

---

## Problem
Over 1,000 sanitation workers died in sewer and septic tank 
incidents across India between 2016–2023. Workers enter confined 
underground spaces with zero real-time monitoring or alert system.

## Solution
An IoT-enabled smart helmet that monitors hazardous gas, temperature, 
and worker location — streaming live data to a supervisor dashboard 
with automatic alerts and 3-stage escalation.

## Hardware
- ESP32 Microcontroller
- MQ135 Gas Sensor
- DHT11 Temperature & Humidity
- NEO-6M GPS Module
- Buzzer + LED Indicators

## Tech Stack
- **Frontend:** React (Vite)
- **Backend:** Node.js + Express
- **Database:** Firebase Realtime Database
- **Hardware:** ESP32 Arduino
